# Franco Osorio · Portfolio

Portafolio bilingüe (ES/EN) con tema claro/oscuro, PWA y animación por fotogramas ligada al scroll.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Motion
- Phosphor Icons

## Desarrollo

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run lint
npm run typecheck
npm run build
```

## Contenido

Textos y proyectos editables en:

- `src/content/es.ts`
- `src/content/en.ts`
- `src/content/projects.ts`

## Rutas

- `/es`
- `/en`
